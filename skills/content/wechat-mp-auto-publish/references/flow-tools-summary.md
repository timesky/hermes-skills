## 二、定时任务

| 时间 | 任务 | Job ID | 功能 |
|------|------|--------|------|
| **11:00** | 文章数据分析 | `92e49e93dcd4` | 分析近15天数据，输出创作建议 |
| **17:00** | 主题调研推荐 | `0131aab79846` | 搜索热点，推荐主题到飞书 |

## 三、阶段划分

### 阶段 A：数据分析（11:00，自动）

| 步骤 | 内容 | 工具 | 输出 |
|------|------|------|------|
| 1. 获取数据 | 15天内已发布文章统计 | wechatpy datacube | 阅读量、分享、收藏 |
| 2. 分析热门 | TOP3文章特点分析 | Ollama | 热门特征 |
| 3. 改写对比 | 原素材vs改写效果 | Ollama | 改进建议 |
| 4. 输出建议 | 推送到飞书 | cronjob deliver | 创作建议报告 |

### 阶段 B：调研推荐（17:00，自动）

| 步骤 | 内容 | 工具 | 输出 |
|------|------|------|------|
| 1. 定时触发 | 每天17:00 | cronjob | 启动任务 |
| 2. 热点调研 | 搜索今日热点 | Tavily/Ollama | 热点列表 |
| 3. 结合数据 | 参考分析建议 | Ollama | 优化推荐 |
| 4. 推荐输出 | 发送推荐到飞书 | cronjob deliver | 主题推荐消息 |

### 阶段 C：人工确认（唯一交互点）

用户回复主题序号（如"1"或"1,2,3"）

### 阶段 D：内容生成（自动）

| 步骤 | 内容 | 工具 | 输出 |
|------|------|------|------|
| 1. 深度调研 | 调研确认主题 | Ollama | 调研笔记 |
| 2. 文章生成 | 1500+字，人设风格 | Ollama + Qwen | 文章内容 |
| 3. 配图生成 | 3-5张主题配图 | Stable Diffusion | 图片文件 |
| 4. HTML格式化 | 公众号样式 | 内置模板 | HTML内容 |

### 阶段 E：发布草稿（自动）

| 步骤 | 内容 | 工具 | 输出 |
|------|------|------|------|
| 1. 上传封面 | 永久素材 | wechatpy API | media_id |
| 2. 上传配图 | 永久素材 | wechatpy API | 图片URL |
| 3. 创建草稿 | 填写所有字段 | wechatpy API | draft_id |
| 4. 输出结果 | 发送通知 | cronjob deliver | 发布成功消息 |

## 四、数据分析接口

### 微信公众号数据API

| 接口 | 功能 | 返回数据 |
|------|------|----------|
| `freepublish/batchget` | 已发布文章列表 | article_id, title |
| `datacube/getarticlesummary` | 图文阅读次数 | int_page_read_count, share_count |
| `datacube/getarticletotal` | 图文阅读人数 | int_page_read_user, share_user |
| `datacube/getuserread` | 用户阅读数据 | read_count, read_user |

### 数据指标说明

| 指标 | API字段 | 说明 | 用途 |
|------|----------|------|------|
| **阅读量** | int_page_read_count | 图文页阅读次数 | 热门判断 |
| **阅读人数** | int_page_read_user | 图文页阅读人数 | 完读率估算 |
| **分享次数** | share_count | 分享次数 | 传播力 |
| **分享人数** | share_user | 分享人数 | 传播广度 |
| **收藏次数** | add_to_fav_count | 收藏次数 | 内容价值 |
| **完读率** | (估算) | 阅读人数/点击人数 | 内容质量 |

### 重点分析内容

1. **热门文章TOP3**
   - 阅读量 > 500 的文章
   - 分析主题特点、标题风格、内容结构

2. **改写效果对比**
   - 原素材阅读高 + 改写后阅读低 → 总结原因
   - 原因可能：标题不够吸引、去AI化不够、配图不相关

3. **主题类型分析**
   - 科技类平均阅读 vs 商业类平均阅读
   - 热门关键词提取

4. **创作建议输出**
   - 推荐主题（预计阅读量）
   - 标题风格建议
   - 内容优化建议

## 七、关键参数

### 文章参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 字数 | 1500-2000字 | 按字数计算 |
| 配图数量 | 3-5张 | 符合主题，无商标 |
| 章节数 | 5-6个 | 加开头和结尾 |
| 人设 | 按主题类型 | 科技/商业/生活/热点 |

### 技术参数

| 参数 | 值 | 说明 |
|------|-----|------|
| JSON编码 | ensure_ascii=False | 必需！ |
| 标题长度 | ≤64字符 | 字符数 |
| 摘要长度 | ≤120字符 | 字符数 |
| 封面尺寸 | 900×500 | 推荐 |
| 配图尺寸 | 800×400 | 推荐 |
| 数据范围 | 15天 | 分析窗口 |

## 八、错误处理

| 错误码 | 问题 | 解决方案 |
|--------|------|----------|
| 45003 | 标题超限 | 缩短标题或用ensure_ascii=False |
| 40001 | Token无效 | 即时获取，不缓存 |
| 40007 | media_id无效 | 用永久素材接口 |
| 404 | 数据接口无数据 | 先发布文章积累数据 |
